# Maersk Remote Work Portal - Installation Instructions

## Example SharePoint Integration

**Demo SharePoint Site**: https://thecozm.sharepoint.com/sites/PeopleFunction/

This is a demonstration People Function SharePoint site that shows how the Maersk Remote Work Portal can be integrated into an existing SharePoint environment.

---

## Quick Integration Steps

The remote work portal can be integrated into any SharePoint hero tile in **under 5 minutes**:

1. Navigate to your SharePoint page
2. Click the "Edit details" icon on the hero tile
3. Scroll to "Link to an image, page, document, or video" and click "Change"
4. Select "From a link" and paste your portal URL
5. Click "Republish" to make changes live
